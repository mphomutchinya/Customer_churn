
# Customer churn probability using machine learning

This project was intended to help a business predict the probability of a customer leaving given data.

Forseeing customer churn is an incredibly powerful tool as it can help the retentions team to perform better in retaining clients who have a higher chance of leaving. 

The model used was extreme gradient boosting, which is an ensembling method.

# Evaluation

The model was evaluated on a Kaggle competition using AUC between the predicted probability and the observed target using unseen data, producing a score of: 0.90094.
